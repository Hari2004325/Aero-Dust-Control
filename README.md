# Aero-Dust Control

Aero-Dust Control is a web-based dashboard for visualizing dust mitigation insights around the Raipur-Bhilai heavy vehicle corridor. It highlights air-quality stress points, traffic congestion, and health exposure impacts.

## Live Demo

Open the website here:
https://hari2004325.github.io/Aero-Dust-Control/

## Project Files

- index.html — main page structure
- style.css — visual styling and layout
- app.js — dashboard interactions and dynamic data

## Features

- Interactive hotspot diagnostics
- AQI and dust exposure metrics
- Regional health risk visualization
- Responsive dashboard layout
